## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/LucaBreebaart/Radian_Backend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the client-server in `radian_backend`
   ```sh
   npm start
   ```
4. Start the backend server 
   ```sh
   npm run dev
   ```
